# Cognosphere/Hoyoverse Fan Website

This project is a web application built with Angular, showcasing the games and services of the company Cognosphere, also known as Hoyoverse. It is intended as a fan-made portfolio or a template for a company website.

The website features a modern, responsive design and includes sections for:
*   **Home:** A landing page with a hero video and an overview of the company's products.
*   **Products:** Showcases games like Genshin Impact, Honkai: Star Rail, Zenless Zone Zero, and Honkai Impact 3rd.
*   **Services:** Highlights related services such as HoYoLAB, HOYO-MiX, and N0va Desktop.
*   **About:** A page to describe the company.
*   **Employee List:** A page to list team members.

## Technologies Used

*   [Angular](https.angular.io/)
*   [TypeScript](https://www.typescriptlang.org/)
*   [Tailwind CSS](https://tailwindcss.com/)
